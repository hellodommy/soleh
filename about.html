About soleh!

soleh! is a wordplay between the French word soleil (pronounced so-lay, meaning sun) and the Singlish word leh.

It provides six 2-hour weather forecasts for any day in Singapore by region.

If you have any suggestions, raise an issue or make a pull request on the repository.
